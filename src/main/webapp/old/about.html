О проекте

Цель проекта

Создание интерактивного пособия по изучению ИИ. Связать имеющуюся информацию по ИИ (аудио, видео, тексты, схемы, статьи) в единую систему с возможностью:

  • доступа к множеству разнородной информации через единый удобный интерфейс;
  • быстрого поиска материала по заданным критериям;
  • обсуждения материала пользователями (вопросы, ответы, мнения, статьи)

Возможности

На данный момент реализованы следующие возможности:

  • чтение ИИ;
  • поиск по тексту словосочетаний;
  • поиск определения ИИ терминов;

Техническая поддержка

Email, Skype
